As usual, every three months — a new Thunder Engine release. Today I'm happy to share the latest version of the engine with you!

**What's new?**

This release is packed with features. Here are the key changes:

**Animation**

- Application background state check
- State Blending — an important step toward smooth transitions
- Animator refactoring and bone-related bug fixes

**Editor**

- Custom Property Editors
- Context menu in the property editor
- Highlighting and deleting connections in graphs
- Sprite transformation tool
- Numerous fixes, including crashes when deleting nodes and renaming assets

**Rendering & Text**

- Bitmap font support (an alternative to SDF for small text)
- Fixed multiple text rendering issues

**Core & Engine**

- Crash backtracer — the call stack is now written to the log!
- Standard system cursors

**Where to download?**

The release is already available:

- Via the link: [https://github.com/thunder-engine/thunder/releases/tag/2026.2](https://github.com/thunder-engine/thunder/releases/tag/2026.2)
- Or through **Thunder Hub** — our new launcher application

Thank you for sticking with the project! Installation, testing, and feedback greatly help development.
